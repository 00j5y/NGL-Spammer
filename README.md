### 1- Clone the code in a folder
```bash
git clone https://github.com/00j5y/NGL-Spammer.git
```

### 2- Run a CMD in the folder and download all the libraries
```bash
pip install -r requirements.txt
```

### 3- Run the main in the folder/CMD 
```bash
py main.py
```
